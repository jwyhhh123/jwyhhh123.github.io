如何理解计算机?

post @ 2019-06-18

如果直观地理解计算机，它是按照人类大脑模型重新设计出来的机器。它拥有内存(Memory)，图像显示(graphic)，逻辑门(logic gates),以及Input/Output设备。

不要认为只有个人电脑，台式机属于计算机。计算机包括手机，ipad，ps4以及其他智能设备。

当下的计算机结构基于冯·诺伊曼结构(CPU和存储器分离)，但这种结构并不完美。本质上计算机是对Turing Machine的简单实现，因为它只关心输入和输出。

  • 计算机中央处理器

值得注意地是，我们需要将这些硬件连接到一起从而组成一个完整的计算机，这将需要大量的电路去实现它们之间的连接。所以，我们使用主板(motherboard)将这些元件接合。

Version control

def : 版本控制关注的是文件，代码。当个人/团队需要完成一项工程时，我们关注的是这项工程中的每份文件的进度。通过版本控制系统，每次对内容上的修改都会通过以下方式记录:

Informed Search

post @ 2019-06-02

Informed Search

def : unlike uninformed searching algorithm, informed search carefully choose which node should be visited first in order to minimise the cost.
 
Heuristic functions(启发函数) : h(x) estimate the cost of the cheapest path from the node reached to a goal node.

A star Search

def : best-first informed search algorithm
 
Evaluation function : f(node) = g(node) + h(node)
g(node) : the cost from root node to current node.
 
Algorithm :

  1. visit the node with the smallest f(node) first
  2. placing its children in the frontier
  3. do not place children in the frontier if they are already visited or are in the frontier.
  4. when found the goal node, stop.

Consistency

Heuristic is said to be consistent if h(n) satisfies : h(n) ≤ cost(n, n’) + h(n’) for every edge (n, n’) of the graph. In such a case, A star is complete and admissible.
 
Time.compx : usually exponential
Space.compx : usually exponential

Uninformed Search

post @ 2019-06-02

Problem Formulation

  1. initial state
  2. goal state
  3. possible actions and their effects on states
  4. cost function

Breadth First Search: Queue

Complete: guaranted to find a goal state if one exits and terminate if failed.
Admissible(可采纳的): able to find the shortest path.
Space.Compx = O(b^d - 1) = O(b^d)
Time.Compx = O(b^d -1) = O(b^d)
b: branching factor (二叉 : 2 ; 三叉 : 3)
d: depth of the goal

Depth First Search: Stack

Complete: if the state space is finite,…(incomplete if infinite)
Not admissible: not guaranteed to find the shortest path to a goal
Space.compx = O(b^m)
Time.Compx = O(b^m)
b: branching factor
m: maximum depth of the search tree

NOTES:

  • If a state can be reached by two nodes, we only remember the node associated to the lowest cost.
  • in real practice, it is not the case we can simply find the shortest path, we need to explore the entire maze before.
  • we mark explored nodes in a visited list, and nodes in the frontier is stored in an unvisited list.

Analysing Searching Algorithms

Completeness : if it is guaranteed to find a solution when at least one solution exists.
Admissibility : if it is guaranteed to find the best solution when there are more than one solution.
Time Complexity : the order of computation during the search process in the worst case
Space Complexity : storage space required in the worst case
