Unary operators

2) The unary operator * is called indirection or dereferencing operator. It is applied to a pointer to access the object the pointer points to.

Pointing to different types

A pointer is pointing to other data which has same type of the pointer itself. To be more clear, a pointer has type int should point to a data with type int.

Example:

Representation:

  • If p has type char, then *p = DD on a little-endian computer.
  • If p has type int, then *p = DDCCBBAA on a little-endian computer.
  • If p has type long long which has 8 bytes long, then *p = DDCCBBAA???? where the question marks mean out of bound.
