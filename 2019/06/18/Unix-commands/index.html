MacOS/ basic commands

WY J
2019-06-18 (Updated: 2019-10-07)
Terminal

终端(Terminal)是可以进行人机交互的Input/Output程序。当下主流操作系统 i.e. MacOS, Linux使用虚拟终端(Virtual Terminal), 并通过指令(commands)来实现一些方便用户的操作。
 
这里介绍MacOS环境下的终端指令。需要注意的是MacOS和Linux由Unix操作系统派生，所以终端指令较为相似。

Terminology

Directories: 目录; 对于计算机，将目录的路径理解为指针(pointers)更为合适。

更直观地，我们可以使用树(Tree)表示:

指令格式: command [options…] [arguments…]
command: 指令名称
options: 修饰(modifiers); 通常为 “-“
arguments: 参数(parameters); 通常为文件名称

NOTE : 中括号可以省略。

man 使用手册

def: 对于任意一条命令，我们也许不知道它的具体功能和使用方法。man命令就像一本字典，我们可以查询一项命令的用户手册。
format: man [-k] name
 
example: man pwd

List of some commands

pwd 打印当前路径

cd 更改路径

ls 列出目录下文件

mkdir 新建文件夹

rmdir 删除文件夹

cp 复制文件

find 查找文件

cat 打印文件内容

more cat2.0

diff 比较两个文件

wc 字数统计

mail 邮件

talk 聊天

say 让电脑说话

cal 日历

date 日期

time 时间

ssh 安全外壳协议
