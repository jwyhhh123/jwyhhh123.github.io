Terminal 终端

终端(Terminal)是可以进行人机交互的Input/Output设备。当下主流操作系统 i.e. MacOS, Linux使用虚拟终端(Virtual Terminal)，并通过指令(commands)来实现Input和Output。
 
这里介绍MacOS环境下的终端指令。需要注意的是MacOS和Linux由Unix操作系统派生，所以终端指令较为相似。

Terminology

Directories : 目录; 对于计算机，将目录的路径理解为指针(pointers)更为合适。

更直观地，我们可以使用树(Tree)表示:

指令格式: command [options…] [arguments…]
command : 指令名称
options : 修饰(modifiers); 通常为 “-“
arguments : 参数(parameters); 通常为文件名称

NOTE : 括号可以省略。
