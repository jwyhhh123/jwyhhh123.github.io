The von Neumann Architecture

def: this is the principle of stored-programming computer in which no re-programming is required. Data are accessed from disks.

Von Neumann architecture consists of three main components:

  1. Central Processing Unit (CPU)
  2. Memory
  3. Input/Output interfaces

Processes are activities of computer programs that are used to:

  1. read memory
  2. send data to CPU to compute
  3. communicate with I/O interface
  4. (re)write data
    …

CPU

CPU is the head of a computing system. It includes:

  1. Control Unit (CU)
  2. Arithmetic and Logic Unit (ALU)

Control Unit (CU)
Control Unit performs the following tasks:

  1. Determines the order in which instructions should be executed
  2. Instructs ALU to perform the required operation
  3. Controls which Register to read and write
  4. Controls read and write of Memory
  5. Controls access of Input/ Output devices

NOTE_2: within the Control Unit, there is a circuit counting times. The timing of operations ensures step-by-step computation.

Registers
def: Registers are small storage which are located next to ALU. They are used to store temporary storage during computation. ALU can read and write to registers very fast.

Advantage: they improve the processing speed when compution needs to reuse some temporary data.
for example, to compute c = (a * b) + (a + b) + (b - a) we need to compute:

  1. c = a * b
  2. c = c + (a + b)
  3. c = c + (b - a)

as c is stored in one unit of the register, we save time from reading and writing c again and again.

Input/ Output Interfaces

def: The I/O interfaces are used to receive or send information from and to connected devices. Connected devices are called peripheral devices.

Comparison between GPU and CPU

CPU handles processes concurrently. computation of each process is done in a serial way, namely, one by one.
 
GPU, from the other hand, is good at processing in a parallel way. It contains multiple shader cores in which ALUs are embedded.

Nowadays, machine learning, computer vision and other fields in AI has been using GPUs to deal with high computational tasks by taking the advatange of parallelism.
