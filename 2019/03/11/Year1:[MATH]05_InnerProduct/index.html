Translating to equational representations

Lines :
Given PR of a line : X = P + s v = (p1, p2) + s (v1, v2)
and we look for an ER : a x1 + b x2 = d

  1. a = -v2
  2. b = v1
  3. d = a p1 + b p2

proof :

Planes :
Given PR of a plane : X = P + s · v + t · w = (p1, p2, p3) + s · (v1, v2, v3) + t · (w1, w2, w3)
and we look for an ER : a · x1 + b · x2 + c · x3 = d

  1. a = v2 · w3 − v3 · w2
  2. b = v3 · w1 − v1 · w3
  3. c = v1 · w2 − v2 · w1
  4. d = a · p1 + b · p2 + c · p3

The geometric interpretation of equational presentations

Given a · x1 + b · x2 + c · x3 = d,
normal vector n = (a, b, c) (orthogonal to the plane)
d: negative -> the origin is on the same side of the plane as the normal.
d: positive -> the origin is on the other side of the plane as the normal.
