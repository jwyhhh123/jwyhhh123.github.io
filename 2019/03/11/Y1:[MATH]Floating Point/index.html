A difference between Z and Zm

Z is an ordered ring (0,1,2,3,4,5…)
Zm is NOT an ordered ring (Z3 : 0,1,2,0,1,2..)

The rational numbers (Q)

To check two fractions are equal : a/b = c/d iff a x d = b x c

multiplicative inverse : a * b = 1 mod m
Field : ring law + multiplicative inverse (for each element)

The Euclidean Algorithm (欧几里得算法)

To save space and time in a computer which is operating on fractions, we need to reduce the number by deviding it with a common factor.
 
largest common factor : lcf

validity check : lcf(x, y) = lcf(y, r)
