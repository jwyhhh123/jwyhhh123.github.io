Beyond Regular Languages

Non-regular languages

def : we know there are some languages which are not regular because there are uncountably many languages and countably many regexps.
 
prove that a language is not regular: example of well-bracketed words

  1. if L is regular, then there is a DFA(X, p, δ, Acc) recognizes it.(K’s T)
  2. suppose a, b ∈ L and m < n.
  3. let Xm be the state reached when we start from p and read a^m.
  4. let Xn be the state reached when we start from p and read a^n.
  5. if starting from Xm and reading b^m then we reach a Acc.
  6. if starting from Xn and reading b^m then we reach a nonAcc.
  7. So Xn != Xm
  8. So there are infinitely many states.
  9. we raise a contradiction. Hence the language is not regular.

NOTE : 我们通过这种方式建立了一个DFA返回yes或no，然而论证中出现infinite，和我们的finite矛盾。
