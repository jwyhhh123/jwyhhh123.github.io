Question 1:
Insert numbers{7,1,4,5} in an empty queue. Represent that process using standard constructors .push and .EmptyQueue.

Show, in the standard two-cell notation, the resulting queue

What is the result of the operation .top?

What is the result of the operation .pop?

What is the result of the operation .pop followed by .pop followed by .top on the original queue you created?

Question 3:
Write a recursive procedure .equalList(L1,L2) that returns true if the lists L1 and L2 are the same, and false if they are not.The only procedures it may call are the standard primitive list operators first, rest and isEmpty.

What is the time.Compx?

Question 4:
Given operators first() , rest() and isEmpty(). Write a recursive procedure .member(x, S1) that returns true if item x is in S1 and false if not.

What is the average-case time.Compx and worst-case time.Compx?

Given operators makeList(), first() , rest() , isEmpty() and member(). Write a recursive procedure .intersection(S1, S2) that returns a new linked list which stores all interseted items in S1 and S2.

2) write .avevalue(qt) which returns the average gray-scale value across the whole picture.
