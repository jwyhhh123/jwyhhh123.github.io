Breadth-first search and Depth-first search

BFS: Queue
Complete: guaranted to find a goal state if one exits and terminate if failed.
Admissible(可采纳的): able to find the shortest path.
Space.Compx = O(b^d - 1) = O(b^d)
Time.Compx = O(b^d -1) = O(b^d)
b: branching factor
d: depth of the goal
 
DFS: Stack
Complete: if the state space is finite,…(incomplete if infinite)
Not admissible: not guaranteed to find the shortest path to a goal
Space.compx = O(b^m)
Time.Compx = O(b^m)
b: branching factor
m: maximum depth of the search tree
 

in real practice, it is not the case we can simply find the shortest path, we need to explore the entire maze before. During this process, we are able to build a search tree.

Optimization2

linear regression: trying to find a trend. to find the best fit line

  • .hypothesis h(x) = w(0) + w(1)x(1) + … + w(n)x(n) = w ^T x // assume x(0) = 1
  • .cost function: J(w) = Σ( y(i) - hw(x(i)) )^2 // w is the value to found actual h(w)
  • .(partial) differential(w0 w1) :
    • w1 = w1 + α . ( y(j)i - hw( x(j)i ) ) . x(j)i
    • w0 = w0 + α . ( yj - hw( x(j)0 ) ) . 1

  logistic regression
  - classification: trying to find a boundary to differentiate two classes.

  • dependant variable {0,1} (someone has cancer or hasn’t)
  • .sigmoid function: g(z) = 1 / ( 1 + e^(-z) )
  • .cost function:
    • J(w) = -1/m . Σ[ y(i)log( hw(x(i))) + (1-y(i))(log(1-hw(x(i)))] // 1/m is to take the average of m sums
  • .(partial) differential(w0 w1) :
    • w1 = w1 + α . ( y(j)i - hw( x(j)i ) ) . x(j)i
    • w0 = w0 + α . ( yj - hw( x(j)0 ) ) . 1
  • decision boundary
