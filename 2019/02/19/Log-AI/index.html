:Optimization2

 linear regression: trying to find a trend. to find the best fit line

  • .hypothesis h(x) = w(0) + w(1)x(1) + … + w(n)x(n) = w ^T x // assume x(0) = 1
  • .cost function: J(w) = Σ( y(i) - hw(x(i)) )^2 // w is the value to found actual h(w)
  • .(partial) differential(w0 w1) :
    • w1 = w1 + α . ( y(j)i - hw( x(j)i ) ) . x(j)i
    • w0 = w0 + α . ( yj - hw( x(j)0 ) ) . 1

  logistic regression
  - classification: trying to find a boundary to differentiate two classes.

  • dependant variable {0,1} (someone has cancer or hasn’t)

  • .sigmoid function: g(z) = 1 / ( 1 + e^(-z) )

  • linear separable
  • cost function is not convex(代价函数不是凸函数,凸函数，有极值)
  • .cost function:
    • J(w) = -1/m . Σ[ y(i)log( hw(x(i))) + (1-y(i))(log(1-hw(x(i)))] // 1/m is to take the average of m sums
  • .(partial) differential(w0 w1) :
    • w1 = w1 + α . ( y(j)i - hw( x(j)i ) ) . x(j)i
    • w0 = w0 + α . ( yj - hw( x(j)0 ) ) . 1
